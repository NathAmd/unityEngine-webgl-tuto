Création d’un mini-jeu

Scène :

Dans l’onglet “Scènes”, créez une nouvelle scène en cliquant sur “Ajouter une scène”.

Ajoutez des objets :

Ajoutez des objets 3D (comme des cubes, sphères, etc.) à votre scène.

Scripts :

Créez des scripts pour gérer le comportement de vos objets (mouvement, interaction, etc.). Utilisez le langage C#.

Testez votre jeu :

Testez votre jeu en mode “Jouer” dans Unity.
